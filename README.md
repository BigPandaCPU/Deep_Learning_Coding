# Deep_Learning_Coding
Numpy实现DNN与CNN网络，

```
|-Deep_Learning_Coding
	|-data 			存放mnist数据
	|-docs 			主要包含相关公式推导
	|-images		说明文档插入图片
	|-nn			核心代码
	|-Conv.py		卷积神经网络实现
	|-mlp.py		全连接网络实现
	|-dataset.py		加载mnist数据集
	|-main.py		nn模块测试样例
```

git clone https://github.com/hui126/Deep_Learning_Coding.git
